## üé® How to View

### Option 1: GitHub/GitLab
These diagrams use Mermaid syntax and will render automatically on GitHub/GitLab when viewing the .md files.

### Option 2: Online Mermaid Editor
1. Go to [Mermaid Live Editor](https://mermaid.live/)
2. Copy the code from any diagram file
3. Paste it into the editor
4. View the rendered diagram

### Option 3: VS Code Extension
1. Install "Markdown Preview Mermaid Support" extension
2. Open the .md file
3. Use Markdown Preview (Ctrl+Shift+V) to view diagrams
